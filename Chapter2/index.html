Bold and Strong

'b' tag is used to create a 'b' element, which represents bold text in an HTML document.
'strong' tag is used to create a 'strong' element, which represents strong text in an HTML document.

Italic and Emphasis

'i' tag is used to create a 'i' element, which represents italic text in an HTML document.
'em' tag is used to create a 'em' element, which represents emphasis text in an HTML document.

HTML Abbreviation

HTML Hypertext Markup Language.

Citations

In HTML, "citations" refer to when you mention a quote, reference, or any other source. It is used to reference academic or reliable sources. In HTML, the `< cite >` tag is used for this purpose, representing the title of a work or the name of an author. This tag typically displays the text in italic format, but you can style it with CSS.

Example:

One of my favorite novel is The Prisoner Of Zenda by Anthony Hope.

In this, "The Prisoner Of Zenda" is italicized as a citation..

Definitions

In HTML, the `< dfn >` (definition) tag is used to mark the definition of a term or concept. When a specific word is defined, it is wrapped inside the `< dfn >` tag. The browser highlights it to indicate that it's a defined term.

Example:

HTML stands for HyperText Markup Language.

In this example, "HTML" is being defined.
